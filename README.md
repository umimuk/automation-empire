# automation-empire — 自動化帝国

AI×副業をテーマにした経営シミュレーションゲーム（カイロソフト系）。
ポンコツAIエージェントを雇って副業に配置し、やらかしを乗り越えながら「自動化帝国」を築くゲーム。

## 遊べる場所

https://umimuk.github.io/automation-empire/

## 現在の状態

- 開発中（Phase 5まで完了・GitHub Pagesで公開中）
- 実装済み機能:
  - 御三家（ポエム/バグ丸/ハッタリ）選択システム
  - ターン制ループ（1ターン=1週間）
  - 副業選択・実行（Lv1〜Lv5の21種）
  - 設備購入・事務所5段階拡張
  - やらかしイベント（13種×4選択肢）
  - 月次レポート・確定申告イベント
  - 評判システム（E〜Sランク）
  - エンディング分岐5種
  - 隠し称号10種
  - シナジーボーナス6組
